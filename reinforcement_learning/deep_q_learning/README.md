# Deep Q-learning

### Entraine un modèle de Deep Q-Learning à jouer à breakout.

- [setup.sh](/setup.sh) initialise et prépare un environnement **pyenv** pour entrainer le modèle.

- [requirements.txt](/requirements.txt) liste les librairies nécessaires au fonctionnement du code.

- [train.py](/train.py) entraine le modèle sur 1.5M de steps (faire plus dégrade les perfs).

- [play.py](/play.py) permet de visualiser les performances du modèle via l'interface du jeu.
